## 📌 1. What is the Service Container in Laravel and how does it work?

> **💡 Short Answer**
>
> The Service Container manages class dependencies and performs dependency injection automatically. It resolves interfaces and injects concrete implementations whenever required, making Laravel applications loosely coupled, testable, and maintainable.

> **💡 Short Answer**
>

> **💡 Short Answer**

## 📌 4. Lazy Loading vs Eager Loading?

> **💡 Short Answer**
>
> Lazy Loading fetches relationships only when accessed, whereas Eager Loading retrieves related records upfront using `with()`, preventing the N+1 query problem.

> **💡 Short Answer**
